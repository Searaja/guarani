Here you can find the files, codes and supplementary information used in "Neurocognitive markers of word integration in a second language".
The zip files from 001-004 to 021-024 contain the .fif files obtained in the experiment.
The file word_pairs_info.xlsx contains the information used to make the Supplementary_Table_2.pdf and some more information about the stimuli.
The file subject_info.xlsx contains the information used to make the Supplementary_Table_2.pdf and some more information about the subjects.
The code.zip contains the code (3 .ipynb files) used to process the data from the .fif subject files mentioned above and behavioral files (saved in code.zip as well).
